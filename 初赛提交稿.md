## 1. 创意名称 + 创意介绍

**创意名称**：xx memory（通用智能记忆助手）

**想解决什么问题**：
学生、考研党和职场人士每天都要记忆大量知识点，但传统背书记忆效率低、容易遗忘、缺乏科学规划。市面上虽有 Anki、百词斩等工具，但要么学习曲线陡峭，要么只局限于单词，无法灵活适配专业课、法律条文、医学知识等多元化内容。

**为什么会想到做这个**：
作为备考者，作者深受「背了忘、忘了背」的困扰。希望有一款工具能把任意学习资料（Markdown / PDF / Word / Excel）一键转化为科学排布的复习卡片，并用艾宾浩斯遗忘曲线自动规划复习节奏，像私人记忆教练一样高效管理记忆负担。

**大概是什么产品**：
一款基于艾宾浩斯遗忘曲线与间隔重复算法的 Android 智能记忆应用，支持多格式文档导入 → 自动提取知识点 → 生成问答卡片 → 科学规划复习 → 数据可视化统计。

## 2. 目标用户及痛点

**面向哪些用户**：
- 在校学生（大学生、考研 / 考公 / 考编群体）
- 语言学习者（英语、日语等）
- 职场专业人士（医生、律师、程序员等需要持续学习考证的人群）
- 专业考试备考者（法考、医考、CPA 等）

**在什么场景下使用**：
- 等公交、排队、午休等碎片化时间随手刷卡片复习
- 课前将 Markdown 笔记导入，自动生成问答卡片
- 考前将 PDF 教材 / Excel 考点整理导入，App 按遗忘曲线规划每日任务

**当前痛点**：
- 手动制作纸质 / 电子卡片耗时耗力
- 不知道何时复习最有效，凭感觉安排容易遗漏
- 现有工具（Anki）操作复杂，新手难以上手
- 缺乏对自己记忆薄弱点的数据洞察

---

## 3. 价值与意义

**效率提升**：
将「文档阅读 → 手动制卡 → 安排复习」三步压缩为「一键导入 → 自动制卡 → 智能规划」，单份 20 页 PDF 的制卡时间从 2 小时缩短到 2 分钟。

**社会价值**：
降低科学记忆方法的使用门槛，让更多人能利用认知科学规律（间隔重复、主动回忆）提升学习效率，尤其适合时间紧张但记忆负担重的备考群体。

## 4. 作品 DEMO 说明

**可运行性**：
本作品为完整的 Android 原生应用，已通过 `assembleDebug` 构建出可安装 APK（约 17.6 MB），在 Android 12 模拟器及真机上验证通过，可正常启动、导入卡片、复习、查看统计。

**核心功能清单**：

| 功能模块 | 完成状态 | 说明 |
|---------|---------|------|
| 多格式导入 | ✅ | Markdown / PDF / Word / Excel / 纯文本 / JSON / URL 全支持 |
| 智能制卡 | ✅ | 自动解析文档内容，提取关键知识点生成问答卡片 |
| 间隔重复算法 | ✅ | 基于艾宾浩斯遗忘曲线 + SM-2 算法，四级评估 |
| 墨水屏模式 | ✅ | 专为墨水屏设备优化的灰度 UI，无动画、低功耗 |
| 学习统计 | ✅ | 环形进度 + 周视图日历 + 卡片组筛选 + Streak 连续天数 |
| 困难优先排序 | ✅ | 智能识别易遗忘卡片，优先安排复习 |
| 通知提醒 | ✅ | 基于卡片到期时间自动调度本地闹钟提醒 |
| AI Skill 支持 | ✅ | 提供 TRAE Skill 文件，AI 可直接完成「解析 → 制卡 → 编排」 |
| 默写模式 | ✅ | 类似百词斩的古诗词全文默写，强化精细编码 |
| BBDC 沉浸式复习 | ✅ | 选错重试 + 错题降级，符合「不背单词」复习流 |

**技术亮点**：
- **MVVM + Repository Pattern** 架构，数据层（Room）与 UI 层（Compose）完全解耦
- 纯 **Jetpack Compose** 声明式 UI，支持动态主题切换与墨水屏适配
- **DifficultFirstSorter** 智能排序算法：综合历史正确率、FSRS difficulty、近期失败记录动态调整复习优先级
- 完整的单元测试覆盖（DifficultFirstSorterTest / JsonCardParserTest / MemoryAlgorithmTest / SchedulerUtilsTest）
- MuMu 模拟器全功能自动化测试通过（20 项测试，19 PASS / 1 INFO / 0 FAIL）

## 5. 快速体验

### 方式一：直接安装 APK
1. 下载 APK：https://github.com/xx8888888-xh/xx-memory/blob/main/releases/xx-memory-v1.0.0-alpha.apk
2. 在 Android 设备上安装并运行
3. 首次启动自动导入 6 张默认卡片（含古诗词默写卡片），可直接体验复习流程

### 方式二：源码构建
```bash
git clone https://github.com/xx8888888-xh/xx-memory.git
cd xx-memory/android
./gradlew assembleDebug
# APK 位于 app/build/outputs/apk/debug/app-debug.apk
```

## 6. 使用 TRAE 的过程

本项目全程使用 **TRAE IDE** 进行开发，AI 辅助完成了以下工作：
- 项目脚手架搭建（MVVM 架构、Navigation、Room 数据库）
- Jetpack Compose UI 组件生成（卡片翻转、进度环、统计图表）

- 文档解析模块（Apache POI、PDFBox 集成）
- MuMu 模拟器自动化测试脚本编写
- 单元测试与对抗式审查（CRITIC Agent 逻辑漏洞检查）

## 7. 后续规划（复赛目标）

若晋级复赛，计划进一步完善：
- **手写输入支持**：支持用户在默写卡片上手写输入，增强肌肉记忆
