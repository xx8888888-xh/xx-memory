## 万一遇到问题

| 问题 | 应对 |
|------|------|
| Markdown 图片不显示 | 社区一般会自动解析 raw.githubusercontent.com 链接；不显示就把图片下载后用社区「上传图片」功能重新插图 |
| APK 无法直接下载 | 在 GitHub 仓库的 release 页面直接拖拽下载，或用 `adb install` 推送 |
| 帖子被审核驳回 | 检查是否带 `#TRAE AI 创造力大赛` `#vibe coding 大赏` 标签 |
| 不确定是否报名成功 | 每个工作日审核，7.15 前会收到通知 |
