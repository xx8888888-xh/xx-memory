## 一、作品概述

xx memory 是一款基于**艾宾浩斯遗忘曲线**与**间隔重复算法**的 Android 智能记忆应用。它把任意学习资料（Markdown / PDF / Word / Excel / JSON / URL）一键转化为按遗忘曲线科学排布的复习卡片，配合 AI Skill 自动制卡、智能排序、多模态复习，把「背了忘、忘了背」的低效记忆模式升级为「按需复习、最小负担、长期保持」的科学记忆模式。

**目标用户**：在校学生、考研考公群体、语言学习者、医生律师程序员等需要持续学习考证的职场人士、法考 / 医考 / CPA 等专业考试备考者。

## 三、技术亮点

### 1. 科学的记忆算法
- **艾宾浩斯遗忘曲线** + **SM-2** 间隔重复算法
- **FSRS 难度评估**（DifficultFirstSorter.kt）综合三维特征：
  - 历史正确率
  - 内在难度（FSRS difficulty / SM-2 easeFactor）
  - 近期失败记录
- 智能识别「易遗忘卡片」并优先安排复习，提升单次复习 ROI

### 2. 优雅的工程架构
- **MVVM + Repository Pattern**：数据层（Room）与 UI 层（Compose）完全解耦
- **Kotlin Coroutines + Flow**：异步数据流自然背压
- **纯 Jetpack Compose 声明式 UI**：支持动态主题切换
- **KSP 注解处理**：Room 编译期生成 DAO
- **多视角代码审查**：ARCHITECT / EXECUTOR / CRITIC / QA 四角色对抗

### 3. 极致的用户体验
- **默写区最大化**：删除复习进度条，腾出空间给 OutlinedTextField 占满
- **首字母提示**：单字/单词答案避免直接泄露
- **墨水屏适配**：图标最深处仅用灰色，无任何动画
- **详情页精简**：助记信息直接展示，去掉冗余展开
